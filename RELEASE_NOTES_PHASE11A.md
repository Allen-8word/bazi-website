## 部署 SOP(Mac)

1. 打開 Finder,解壓 `bazi-website-v9.zip`
2. 將上列 5 個檔案覆蓋到本機 repo 對應位置
3. 終端機:`git add -A && git commit -m "Phase 11-A: 修仙包裝層(分享卡稱號 + 流年標題)" && git push`
4. Vercel 自動部署後,用手機開分享卡確認修仙區塊顯示
5. 線上煙霧測試:排一個盤 → 進分析報告 → 確認第四段標題為「流 年 天 劫 與 機 緣」、付費按鈕為「渡劫指南」
