桌機版自適應視覺驗證

請調整瀏覽器寬度測試三個斷點的視覺變化
建議用 Mac Safari／Chrome 拖曳視窗邊緣測試

三個視覺斷點
手機 (<768px) → 平板 (768-1199px) → 桌機 (1200px+)

手機 (<768px)
body 背景：--surface-page (#FAF6EF 米色)
內容區：直接貼齊頁面，無陰影

平板 / 一般桌機 (768-1199px)
body 背景變深：--surface-muted (#F0E8D8)
內容區：浮出陰影、維持 560px 寬度居中

大螢幕 (1200px+)
body 背景再加深：#E8E0CE
內容區的「漂浮感」更明顯

當前頁面的視覺呈現

你現在看到這個區塊：

  • 在手機尺寸，背景是淡米色，內容貼齊邊緣
  • 在平板以上，整個內容區會浮出陰影，兩側顯露出較深的米色
  • 在1200px 以上，背景變得更深，陰影對比更明顯

✓ Phase 12 驗收標準

手機（<768px）：背景單一米色，內容直接貼齊頁面
平板（768px+）：背景變深米色（F0E8D8），內容區浮出陰影
大桌機（1200px+）：背景再加深（E8E0CE），對比更明顯
內容區最大寬度始終為 560px（不會在桌機被拉滿）
內容區的陰影是淡的（不要太重，符合 Halo 平面化風格）
拖曳瀏覽器寬度時三種狀態切換流暢
列印 / 另存 PDF 時自動關閉陰影與背景對比
